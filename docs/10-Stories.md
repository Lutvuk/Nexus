# Stories

## Epic 1: Authentication and Access

- As a guest, I can register and log in so I can access workspace data.
- As an authenticated user, I am redirected away from guest-only pages.
- As a user, my protected routes remain inaccessible without valid auth.

## Epic 2: Workspace and Membership

- As an owner, I can create and rename a workspace.
- As an owner/admin, I can invite users and manage member roles.
- As a user, I can accept/decline invitations and join via tokenized links.

## Epic 3: Board and Workflow Core

- As a member, I can create a board in a selected workspace.
- As a member, I can add columns and cards to represent process steps.
- As a member, I can drag cards and columns to reorder workflow.
- As a member, I can archive and restore cards to keep active lanes clean.

## Epic 4: Card Collaboration

- As a member, I can assign users and labels to cards for ownership and classification.
- As a member, I can add checklist items and track completion progress.
- As a member, I can comment on cards to share updates and decisions.
- As a member, I can upload attachments and set cover images for visual context.

## Epic 5: Visibility and Realtime

- As a board participant, I can see updates from other users in near-real-time.
- As a user, I can open activity and notification panels to monitor changes.
- As a user, I can mark notifications as read individually or in bulk.

## Epic 6: Productivity

- As a user, I can create automation rules to reduce repetitive actions.
- As a user, I can define custom fields so cards match team-specific data needs.
- As a user, I can reuse templates to start similar work faster.

## Epic 7: Views and Planning

- As a user, I can switch between board/table/calendar/analytics/planner views.
- As a manager, I can inspect analytics to track throughput and overdue work.

## Epic 8: Board Documentation Notepad

- As a user, I can create board-level notes in board settings.
- As a user, I can edit note title/content and save updates reliably.
- As a user, I can delete obsolete notes.
- As a teammate, I can read shared notes persisted at board level.
- As a user, I see stable footer controls in the editor without overlap.
