## 1. Executive Summary
Project Nexus is a modern, high-performance Kanban board application built to demonstrate full-stack capabilities with **Go (Golang)** and **Angular 18**. The MVP features a buttery-smooth, physics-based drag-and-drop interface, robust backend persistence, and a premium "Glassmorphism" UI design.

## 2. Technology Stack

### Backend (Go)
-   **Framework**: Gin Web Framework (REST API)
-   **Database**: PostgreSQL 15
-   **ORM**: GORM (Object Relational Mapping)
-   **Utilities**: `godotenv` (Config), `uuid` (Identity)
-   **Architecture**: Repository Pattern, Clean Architecture

### Frontend (Angular)
-   **Framework**: Angular 18 (Standalone Components, Signals)
-   **Styling**: Tailwind CSS + Custom CSS Variables (Glassmorphism)
-   **Drag & Drop**: Angular CDK (`@angular/cdk/drag-drop`)
-   **Icons**: Lucide Angular
-   **State Management**: Signal-based local state in Components & Services

### Infrastructure
-   **Containerization**: Docker & Docker Compose
-   **Web Server**: Nginx (for Frontend production build)

## 3. Implemented Features

### 3.1 Kanban Extension (Core)
-   **Physics-Based Drag & Drop**:
    -   Smooth 60fps animations using GPU-accelerated CSS (`will-change: transform`).
    -   Optimistic UI updates (instant movement before server confirmation).
    -   Cross-column transfers and reordering.
-   **Visuals**:
    -   Drag previews lose blur effects for performance.
    -   Drop placeholders show exactly where items will land.

### 3.2 Full Management (CRUD)
-   **Columns**:
    -   **Create**: Glass modal to add new workflow stages.
    -   **Delete**: Confirmation dialog to remove columns (cascading delete).
-   **Cards (Tasks)**:
    -   **Create**: Quick-add items to any column.
    -   **Edit**: Inline editing (Click-to-edit) for titles.
    -   **Delete**: Safe deletion with confirmation.
    -   **Move**: Drag to move (updates `position` and `column_id`).

### 3.3 UI/UX Polish
-   **Glassmorphism Design**: Frosted glass effects, gradients, and vibrant colors.
-   **Global Dialog System**: Custom-built `DialogService` that solves z-index stacking context issues by rendering modals at the root level.
-   **Toast Notifications**: Animated purple toasts for Success/Error feedback.
-   **Mobile Optimization**:
    -   Horizontal Scroll Snapping (`.snap-x`) for native mobile feel.
    -   Touch-optimized targets (>44px).
    -   Overscroll behavior control.

### 3.4 Reliability
-   **Resilient Networking**: `retry(3)` strategy for API calls to handle flaky networks.
-   **Error Handling**: User-friendly error messages via Toast system.
